# 💳 Credit Card Fraud Analytics Dashboard

## 📌 Project Overview

This project analyzes credit card transaction data to identify fraud patterns, transaction trends, customer behavior, and terminal performance. The analysis was performed using SQL, and an interactive dashboard was developed in Power BI to provide meaningful business insights.

---

## 🎯 Project Objectives

- Analyze credit card transaction data
- Detect fraudulent transactions
- Measure fraud percentage and fraud amount
- Compare genuine vs fraudulent transactions
- Identify high-risk customers and terminals
- Build an interactive Power BI dashboard

---

## 🛠️ Tools & Technologies

- SQL (MySQL)
- Power BI
- CSV Dataset

## 🔍 Key Insights

- Fraud Percentage is approximately **3%**.
- Genuine transactions account for approximately **97%** of all transactions.
- Contactless, Swipe, and Chip transactions show different fraud patterns.
- A small number of customers and terminals contribute significantly to fraud.
- Transaction volume varies across different payment networks.
